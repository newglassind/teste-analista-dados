# Teste Técnico — Analista de Dados

Este repositório contém a base de dados fictícia para o teste técnico de Analista de Dados.

Leia primeiro os arquivos `Teste Técnico — Analista de Dados.pdf` e `README\_BASE.md` e depois analise os arquivos da pasta `dados/`.


A entrega deve conter:

1. Dashboard, relatório visual ou análise construída na ferramenta de sua escolha.
2. Relatório executivo com principais conclusões.
3. Lista de dúvidas, premissas e decisões tomadas.
4. Explicação dos tratamentos realizados nos dados.
5. Link ou arquivo da gravação da tela durante a execução do teste.

O objetivo é avaliar o raciocínio analítico, a interpretação dos requisitos, a qualidade das perguntas e a coerência dos indicadores propostos.

Escolha a melhor forma de entrega-los. (isso também é uma avaliação) rsrs
